# RestAPI

### Resource

- 흔히 브라우저의 주소 바에 입력하는 주소를 URL 이라고 함
- URL 은 <b>어떤 것</b>들의 URL 이라고 표현 (예를 들면 홈페이지의 URL 등)

```javascript
GET /stations/1 HTTP // 리소스
```

- Resource 의 표현 타입 : Content-Type
- 서버로부터 응답 받는 결과는 Resource 가 아니라 Representation(표현) 이다.
